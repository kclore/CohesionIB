
Take the database file CohesionibDB.bak
Open SSMS and right click on Databases and Restore Database
Click Device as the source and click the button to the right with three dots
Click Add button and find the database file on your machine and click okay
The connectionString is as follows => "Data Source=localhost;Initial Catalog=CohesionibDB;Integrated Security=True;"
So the instance must be localhost


Open the web api project in vscode dotnet run
Use swagger ui to test end points. Go to http://localhost:5000/swagger/index.html
